## Objective:
The main objective of this day is to perform data analysis on employee and sales data using SQL queries. The goal is to understand how to apply aggregations, filtering, joins, and grouping techniques to get the required outcomes.


## Implementation:
1. Table Creation
Created two tables:employee, sales
2. Data Insertion
Inserted sample data for:
employees across departments
sales transactions
3. Aggregation Queries
Used:
sum(), avg(), count(), max(), min()
4. Filtering with HAVING
5. Joins
Joined: employee + sales tables
6. Advanced Queries
Used: order by, limit

## Key Learnings
Strong understanding of 
1. group by and aggregation functions
2. having vs where
3. join multiple tables
4. Difference between: where, having
