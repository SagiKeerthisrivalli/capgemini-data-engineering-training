Objective :
In this phase, I learnt to perform some complex functions compared to phase 1. This include joining multiple tables and also to perform aggregate functions.

Problem Summary:
We were given 2 tables like customers and orders. And tasks like
1. Combining data from 2 tables
2. Performing aggregate functions

Approach:
I followed this approach:
1. Cleaning data by removing any null values
2. Joined required tables to get required results
3. Applied aggregate functions like sum, avg
4. Generating final results

Transformations used:
• groupBy() with sum(), avg(), count() 
• join() (inner join, left join) 
• filter() for conditions 
• orderBy() for sorting 
• limit()

Output / Results:
• Total money spend per customer 
• Top customers based on total revenue 
• Customers with no orders 
• Average order amount per customer 
• Sorted customer spending data

Challenges Faced:
1. How to join two tables and conditions to apply
2. Syntax of aggregate functions is a bit confusing
